# Nintendo Character Showcase App

A cross-platform mobile application built with Flutter that beautifully showcases popular Nintendo characters. 

## 📱 About the Project

This application serves as an interactive gallery and showcase of iconic Nintendo characters, including Mario, Pikachu, Donkey Kong, Daisy, Toadette (Kinopiko), and Shy Guy. Built entirely with Flutter, it demonstrates responsive UI design and efficient asset management across mobile, web, and desktop platforms. It is similar like the puzzle game.

### ✨ Features
* **Character Gallery:** View high-quality images of fan-favorite Nintendo characters.
* **Cross-Platform Compatibility:** Runs seamlessly on Android, iOS, Web, Windows, macOS, and Linux from a single codebase.
* **Responsive Layout:** Adaptive UI that scales perfectly across different device screen sizes.

## 🛠️ Built With

* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** [Dart](https://dart.dev/)
* **IDE:** Android Studio / VS Code

## 📁 Project Structure

The project directory follows a standard Flutter architecture:

* `lib/`: Contains the main Dart code and UI logic (`main.dart`).
* `assests/images/`: Houses all the local image assets (Mario, Pikachu, etc.) used within the app.
* `android/` & `ios/`: Platform-specific configuration files for mobile deployment.
* `web/`, `macos/`, `linux/`, `windows/`: Platform-specific files for web and desktop deployment.
